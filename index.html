Направления в разработке

Работаю в 4 направлениях: сайт-визитка, трехстраницный личный сайт, лэндингпэйдж, разработка макета сайта.

САЙТ-ВИЗИТКА

Сайт-визитка состоит из одной страницы с краткой информацией и контактными данными с добавлением видео или аудио плееров. Подойдет как для личного сайта, так и для рекламы мероприятия.

ЛИЧНЫЙ САЙТ

Сайт представляет из себя краткий информационный сайт с сылками на ресурсы доступа, контакты. Состоит из 3 страниц: информационная, содержательная и контактная информация.

LANDING PAGE

Landing page или в простонародье лэндинг, оформляется одной длинной странице, на которой находится несколько разделов, с необходимо информацией (ссылками, видео, фотографиями) и контактной информацией.

РАЗРАБОТКА МАКЕТА САЙТА

Разработка макета сайта с сеткой с помощью Figma. Макет можно выполнить для различных устройств.
